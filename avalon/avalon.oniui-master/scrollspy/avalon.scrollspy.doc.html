scrollspy

scrollspy组件

可通过监听元素原生的scroll事件或者结合scrollbar组件使用时，监听其scroll，根据scrollTop，scrollLeft计算判定配置指定的panel list应该切换到什么位置，绑定方式是ms-scrollspy=""，这样可以跟其他组件绑定在一个元素上

使用说明
名字 类型 默认值 说明
配置参数
onInit Member avalon.noop onInit(vmodel, options, vmodels) 完成初始化之后的回调,call as element's interface
onChange Member avalon.noop onChange(index, ele, widgetElement, options) 滚动到应该显示那个tab的index，以及targetListGetter返回的list[index]值，以及绑定scrollspy的元素，options
axis String "y" 滚动条滚动的方向，默认是竖直方向y，取值为x的时候，表示水平方向
targetListGetter(spytarget) Function targetListGetter() 获取tab list函数，元素是数字或者元素id索引，默认是元素id索引
panelListGetter Member avalon.noop panelListGetter() 获取panel list函数，返回一个数组，元素是dom，或者返回空，默认返回空，panel跟targetListGetter返回的数组项，是一一对应的，来实现menu的切换，而内容滚动或切换到对应的panel
panelGetter(id,index,list,options) Function panelGetter(pannelId, pannelIndex, panelsList, options) 获取panel函数，默认返回 panelsList[pannelIndex] || Id = pannelId
spytarget Unary void 0 指定滚动关联的元素id，默认的targetListGetter会到这个指定元素内寻找li元素作为所有的菜单选项，spytarget会作为第一个参数传递给targetListGetter,panelListGetter
接口方法与固有属性
scrollTo(id,index) Function scrollTo(id, index) 滚动到panel位置，滚动到 panelList[index] || dom.id = id的元素的地方，该方法绑定在onInit的返回的options参数上返回，供调用
