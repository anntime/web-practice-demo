mmRouter

Name 路由器

路由器插件，实现类似angular ui-router的功能

	avalon.state("contacts", {
	    controller: "test", // 指定controller
	    abstract: true, // 是否是抽象状态
	    url: "/contacts", // 触发的url
	    templateUrl: "contacts.html", // 模板url
	    onChange: function() { // 回调
	        ...
	    }
	})
